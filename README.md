## Publish on GitHub Pages
This Vite project already uses `base: './'` in `vite.config.ts`, so it can be published under any GitHub Pages repository path.

Recommended simple GitHub Pages deployment:
1. Create a public GitHub repository.
2. Upload all files inside this `app` folder to the repository root.
3. In GitHub, go to Settings → Pages.
4. Choose GitHub Actions as the source.
5. Add a GitHub Pages deploy workflow for Vite, or use your preferred static deployment method.
6. Submit the GitHub Pages URL after confirming it opens in an incognito/private browser.
